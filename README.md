# asteroids-game

A basic version of the classic Asteroids arcade game built using Python and Pygame.

This project was completed as part of the backend development curriculum at **Boot.dev**.

## Requirements

To run this game, you will need:
* **Python 3** (Program written in 3.13.11)
* **Pygame** library

## Controls

* **W / S**: Move Forward / Backward
* **A / D**: Rotate Left / Right
* **Space**: Shoot
